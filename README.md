# passwordchecker
A simple Python-based password strength checker that analyzes a given password and provides feedback on its security level. Useful for learning Python, practicing input validation, and understanding password-strength rules.

# Features

Checks password length

Detects lowercase, uppercase, numbers, and special characters

Calculates an overall strength score

Gives clear feedback on how to improve the password

Fully written in Python
